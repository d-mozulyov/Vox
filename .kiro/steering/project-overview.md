# Обзор проекта Vox

## Описание проекта

Vox — это помощник по вводу текста голосом с открытым исходным кодом. Проект разрабатывается на Go и публикуется на GitHub под лицензией MIT.

## Ключевые особенности

- **Открытый исходный код**: MIT лицензия, доступен для международного сообщества
- **Бесплатность**: Изначально бесплатное решение
- **Интеграция с Mistral AI**: Использование бесплатного API Mistral (voxtral-mini-latest, voxtral-small-latest) через подписку "Experiment"
- **Гибкость**: Поддержка любых OpenAI-совместимых бэкендов через настройки
- **Контекстная транскрибация**: Анализ приложений и контекста для высокого качества распознавания
- **Кроссплатформенность**: Поддержка Windows, Linux, macOS (x64 и arm64)

## Архитектура приложения

- Системный трей-агент, запускающийся при старте ОС
- UI для настройки системы
- Горячие клавиши для управления записью
- Визуальные и звуковые индикаторы состояния
- Фиолетовая цветовая схема (в стиле Kiro)

## Технологический стек

- **Язык**: Go
- **Платформы**: Windows, Linux (musl для совместимости), macOS
- **Архитектуры**: x64, arm64
- **CI/CD**: GitHub Actions
- **Релизы**: GitHub Releases с артефактами и установщиками

## Ключевые направления развития

1. **Контекстная осведомленность**: Анализ активного приложения для улучшения качества транскрибации
2. **Анализ ранее введённого текста**: Использование истории для контекста
3. **Глоссарии**: На уровне приложения и проекта/директории
4. **Проектный контекст**: Хранение и обработка контекста в рамках проекта

## Конкурентные преимущества

- Использование `/v1/chat/completions` вместо классической транскрибации для наполнения контекста
- Интеллектуальный анализ контекста приложения
- Открытый исходный код и бесплатность
- Экономичность (voxtral-mini-latest очень дешёвая модель)
